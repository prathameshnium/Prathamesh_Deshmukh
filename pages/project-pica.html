PICA: Python-based Instrument Control and Automation

PICA is a comprehensive software suite designed to automate laboratory instruments for materials science and condensed matter physics research. It features a central GUI launcher and a modular architecture, using Python and PyVISA to streamline data acquisition and enhance experimental efficiency.
