PICA: Python-based Instrument Control and Automation

A modular software suite for automating laboratory measurements in physics research.

Project Overview

PICA is a software suite designed to provide a robust framework for automating laboratory instruments in materials science and condensed matter physics research.

The suite features a central GUI, the PICA Launcher, which serves as a dashboard for managing and executing a variety of characterization experiments.

A key architectural feature is the use of isolated process execution for each measurement module via Python's `multiprocessing` library, ensuring high stability and preventing inter-script conflicts.

Core Features

Centralized Dashboard

A comprehensive GUI for launching all modules.

Isolated Processes

Ensures stability and prevents resource conflicts.

VISA Instrument Scanner

Discover, identify, and troubleshoot connections.

Modular Architecture

Self-contained modules for each experiment.

Embedded Docs

In-app viewer for README and license files.

System Console Log

Real-time status updates and diagnostics.

Getting Started

1. Prerequisites

  • Python 3.9 or newer.
  • NI-VISA Driver for your operating system.

2. Installation

Clone the repository:

git clone https://github.com/prathameshnium/PICA-Python-Instrument-Control-and-Automation.git
cd PICA-Python-Instrument-Control-and-Automation

Install dependencies:

pip install -r requirements.txt

Launch the application:

python PICA_Launcher.py

Resources & Documentation
